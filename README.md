# DSLocation

### Swift Package Manager

* Installation with Swift Package Manager

```swift
dependencies: [
    .package(url: "https://github.com/swp-song/DSLocation.git", .upToNextMajor(from: "2.0.2"))
]
```

-----

 ### Installation CocoaPods 

 * Installation CocoaPods :

```ruyby
pod search DSLocation
pod 'DSLocation'
```

-----
